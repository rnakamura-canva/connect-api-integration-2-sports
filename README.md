# connect-api-integration-1-java

Spring Boot 3 project with Java 21, JUnit 5, and Mockito.

## Requirements
- Java 21
- Maven 3.6+

## Build and Run
```bash
./mvnw clean install
./mvnw spring-boot:run
```

## Run Tests
```bash
./mvnw test
```

## Dependencies
- Spring Boot 3.4.0
- Java 21
- JUnit 5 (included in spring-boot-starter-test)
- Mockito (included in spring-boot-starter-test)
